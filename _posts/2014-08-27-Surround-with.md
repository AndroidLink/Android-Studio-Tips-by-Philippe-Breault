This can be used to wrap a block of code in some structure. Usually an if statement, a loop, a try-catch or a runnable.

If you have nothing selected, it will wrap the current line.

Shortcut: 

 - Mac: `Cmd+Alt+T`
 - Windows/Linux: `Ctrl+Alt+Tï»¿`
