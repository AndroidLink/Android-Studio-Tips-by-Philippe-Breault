This will contextually expand the current selection.  
E.g. it will select the current variable, then the statement, then the method, etc. 

Shortcut: 

 - Mac: `Alt+Up/Down`
 - Windows/Linux: `Ctrl+W / Ctrl+Shift+Wï»¿`
